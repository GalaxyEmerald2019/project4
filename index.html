Springtime in Michigan

Springtime in Ann Arbor is a beautiful time to walk, notice the flowers, and enjoy the local parks. This site shares a small collection of scenes inspired by those walks.

Use the navigation above to open neighborhood and park galleries. You can also switch the theme and the site language using the buttons in the header.

Why this version is better

  • Images are bundled locally, so they work reliably on GitHub Pages.
  • A skip link helps keyboard users jump to the main content.
  • Theme and language preferences are saved automatically.
